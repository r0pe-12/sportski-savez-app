## Cilj
Browser smoke testovi za sve ključne stranice, end-to-end testovi za 2-3 ključna user journey-a (UC5 cijeli flow, UC8 verifikacija, UC10 rezultati), security audit, performance smoke. **Demo-ready**.

## Izlazi (deliverables)

### Browser smoke (Pest 4 Browser)
- [ ] Smoke test za sve auth stranice (login, register, forgot, reset, verify)
- [ ] Smoke za admin panel index, users, schools, sports, competitions, results, audit-log
- [ ] Smoke za profesor dashboard, teams index, teams/create, profile
- [ ] Smoke za učenik profile, schedule
- [ ] Sve stranice load bez JS console errora

### End-to-end (Pest 4 Browser)
- [ ] **Journey 1 — UC5 full flow:** Profesor login → kreira draft tim → bira sport → dodaje 3 člana → uploaduje 3 potvrde (FakeOcr validira) → potpisuje → submit; Admin login → vidi submitted → odobri; Profesor vidi active status
- [ ] **Journey 2 — UC8:** Admin login → bira učenika → verify → status verified (ili mismatched za drugi učenik)
- [ ] **Journey 3 — UC10:** Admin login → bira completed takmičenje → unosi rezultate za sve ekipe → ekipa status completed; Učenik login → vidi medalju na profilu

### Performance smoke
- [ ] Login response < 500ms (P95)
- [ ] Schedule page (sa 50 takmičenja) load < 1s
- [ ] Audit log paginacija < 500ms (1000 zapisa)
- [ ] Team registration form prikazuje sve sportove i učenike < 300ms

### Security audit (basic)
- [ ] CSRF tokens prisutni na svim form-ovima
- [ ] XSS test: malicious input u team name, school name, signature — escape-uje se
- [ ] SQL injection test (Laravel Eloquent default zaštita; basic provjera kroz Form Request validation)
- [ ] File upload: malicious filename, malicious MIME — odbija se
- [ ] Authorization test: profesor pokušava da pristupi admin ruti → 403
- [ ] Sessions: logout invalidira session

### Final clean-up
- [ ] `php artisan test --compact` zelena
- [ ] `npm run build` bez warnings
- [ ] Pint clean (`vendor/bin/pint --test`)
- [ ] `git tag v1.0` posle merge-a

### Demo skripta
- [ ] `docs/demo-script.md` — korak-po-korak demo za ADIS predaju (od `php artisan migrate && php artisan db:seed` — **ne fresh**, čuvamo `ai_dnevnik_sesije` — do "evo medalja na profilu")
- [ ] Skripta uključuje screenshot opis za svaki demo korak

## Shared edit zones
- `tests/Browser/` ili `tests/Feature/` — feature tests
- `docs/demo-script.md` — novi fajl
- `package.json` ako treba dodati Pest browser testing dependencies (treba se već biti instaliran sa Pest 4)

## Acceptance criteria
- Browser smoke prolazi
- 3 e2e journey-a prolaze
- Performance smoke prolazi
- Security audit basic checklist prošao
- `php artisan test` zelena
- `npm run build` clean
- Demo skripta verifikovana — možeš pratiti i sistem radi

## NE radi
- Nemoj load testing sa hiljadama korisnika (out of scope)
- Nemoj penetration testing (basic security audit je dovoljan)
- Nemoj accessibility audit van WCAG AA basic check-a (spec 8 NFR)
- Nemoj advanced monitoring setup (CloudWatch, Sentry) — log fajlovi su dovoljni
