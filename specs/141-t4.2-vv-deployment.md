## Deliverable 2: Lokalna instalacija

**Pokriva profesorov zahtjev 3a/3b: priprema za puštanje u rad + integracija.**

Korak-po-korak skripta za ocjenjivača (Windows + macOS + Linux):

```markdown
1. git clone <repo>
2. composer install
3. npm install
4. cp .env.example .env && php artisan key:generate
5. touch database/database.sqlite
6. php artisan migrate && php artisan db:seed
7. composer run dev (start Laravel + Vite + queue + Pail u jednom)
8. Otvori http://localhost:8000
9. Login: admin@savez.test / Adm1n!Test (iz seedera)
10. Demo scenario UC5 → UC8 → UC10 (referenca na meta-plan §10)
```

+ troubleshooting: SQLite permissions, Vite manifest, Wayfinder regenerate, queue worker.

## Deliverable 3: Staging + production rollout

**Pokriva profesorov zahtjev 3b: prijedlog kako bi se sistem pustio u rad.**

| Tema | Staging | Production |
|---|---|---|
| Host | Laravel Cloud Hobby | Laravel Cloud Production tier |
| DB | PostgreSQL managed | PostgreSQL managed + read replika |
| Cache/Queue | Redis | Redis + dedicated worker tier |
| Storage | S3 (versioning on) | S3 (versioning + lifecycle 30 days) |
| Email | SES sandbox | SES production |
| OCR | Google Vision (sandbox) | Google Vision (production, rate limited) |
| eDnevnik | Pravi adapter iza feature flag | Pravi adapter, feature flag on |
| Monitoring | Laravel Pulse + Sentry | + CloudWatch + uptime monitoring |
| Pilot | 1 škola, 1 takmičenje | 2-3 škole, postepeni rollout |
